h1. About

A sample DateTime helper class to get time difference in both words and array with example of writing unit test using [SimpleTest](http://www.simpletest.org/en/download.html "SimpleTest")


h3. Time difference as words

<pre><code>$dHelper = new DateTimeHelper();
$dHelper->timeDiffAsWords('2011-03-01 10:00:10');
</code></pre>

h3. Time difference as Array

<pre><code>$dHelper = new DateTimeHelper();
$dHelper->timeDifference('2011-03-01 10:00:10', '2011-03-02 10:00:10')
</code></pre>

h3. Example results

<pre><code>0  - 59 seconds => less than a minute
25 - 35 minutes => half an hour
55 - 65 minutes => an hour
Others          => exactly what should be
</code></pre>

h3. Testing

- Download [SimpleTest](http://www.simpletest.org/en/download.html "SimpleTest") suit and place it to document root directory.
- Run unit_test.php
- Run date_time_helper.test.php
